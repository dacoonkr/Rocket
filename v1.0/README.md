**기본 문법**

```
$1 : 정수 입력 후 1번 스택에 저장
$2 : 실수 입력 후 1번 스택에 저장
$3 : 문자열 입력 후 1번 스택에 저장

% : 1번 스택 맨 위 값 출력
^ : 1번 스택 맨 위 값 파괴

@nm : n번 스택의 맨 위 값을 m번 스택으로 옮김
#nm : n번 스택의 맨 위 값을 m번 스택으로 복제

!1(숫자) : 1번 스택에 정수를 넣음
!2(실수) : 1번 스택에 실수를 넣음
!3(문자열) : 1번 스택에 문자열을 넣음

+ : 1번 스택의 맨 위 2개의 값을 더해 다시 1번 스택에 넣음
- : 1번 스택의 맨 위 2개의 값을 빼 다시 1번 스택에 넣음
* : 1번 스택의 맨 위 2개의 값을 곱해 다시 1번 스택에 넣음
/ : 1번 스택의 맨 위 2개의 값을 나눠 다시 1번 스택에 넣음
: : 1번 스택의 맨 위 2개의 값을 나눈 나머지를 다시 1번 스택에 넣음

&n& : n번째 글자로 이동

~=n* : 1번 스택의 맨 위 2개의 값이 같다면 n번째 글자로 이동 (n에 ], }, >도 가능)
~1n* : 1번 스택의 맨 위 1개의 값이 위에서 2번째의 값보다 크다면 n번째 글자로 이동 (n에 ], }, >도 가능)
~2n* : 1번 스택의 맨 위 1개의 값이 위에서 2번째의 값보다 작다면 n번째 글자로 이동 (n에 ], }, >도 가능)

], }, >: 각각 [, {, < 로 이동
```
