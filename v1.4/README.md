**기본 문법**

```
$1 : 정수 입력 후 1번 스택에 저장
$2 : 실수 입력 후 1번 스택에 저장
$3 : 문자열 입력 후 1번 스택에 저장

% : 1번 스택 맨 위 값 출력
^ : 1번 스택 맨 위 값 파괴

@nm : n번 스택의 맨 위 값을 m번 스택으로 옮김
#nm : n번 스택의 맨 위 값을 m번 스택으로 복제

!1(숫자) : 1번 스택에 정수를 넣음
!2(실수) : 1번 스택에 실수를 넣음
!3(문자열) : 1번 스택에 문자열을 넣음

+ : 1번 스택의 맨 위 2개의 값을 더해 다시 1번 스택에 넣음
- : 1번 스택의 맨 위 2개의 값을 빼 다시 1번 스택에 넣음
* : 1번 스택의 맨 위 2개의 값을 곱해 다시 1번 스택에 넣음
/ : 1번 스택의 맨 위 2개의 값을 나눠 다시 1번 스택에 넣음
: : 1번 스택의 맨 위 2개의 값을 나눈 나머지를 다시 1번 스택에 넣음

&n& : n번째 글자로 이동

~=n* : 1번 스택의 맨 위 2개의 값이 같다면 n번째 글자로 이동 (n에 ], }, >도 가능)
~1n* : 1번 스택의 맨 위 1개의 값이 위에서 2번째의 값보다 크다면 n번째 글자로 이동 (n에 ], }, >도 가능)
~2n* : 1번 스택의 맨 위 1개의 값이 위에서 2번째의 값보다 작다면 n번째 글자로 이동 (n에 ], }, >도 가능)
~-n* : 1번 스택의 맨 위 문자열의 n번째 인덱스의 값을 아스키 코드로 바꿔 1번 스택에 넣는다.

], }, >: 각각 [, {, < 로 이동

?함수명?코드;? : 함수를 선언한다. 함수를 종료할 땐 ;를 쓴다. 재귀함수 쌉가능
"함수명" : 함수를 호출한다.

'라이브러리파일.rk' : library폴더에 있는 라이브러리를 로드한다. 함수 사용하듯이 라이브러리 안 함수를 사용 가능하다.
```

**문자열 처리 라이브러리** `기본` **string.rk**
```
str.length : 문자열 길이 반환 | 문자열 -> 정수

str.front : 맨 앞 글자 반환 | 문자열 -> 문자열

str.back : 맨 뒤 글자 반환 | 문자열 -> 문자열

str.repeat : 문자열을 반복 | 문자열(파괴), 정수(파괴) -> 문자열

str.merge : 두 문자열 합치기 | 문자열(파괴), 문자열(파괴) -> 문자열

ASCtoStr : 아스키 코드를 문자로 | 숫자(파괴) -> 문자열
```
